# To-Do List


// Обычно добавляют информацию не только о функционале, но и о способе
// сборки/компиляции.


Консольное приложение для управления задачами с историей событий.

## Функциональность

- Добавление новых задач
- Просмотр списка всех задач
- Отметка задач как выполненных
- Удаление задач
- Просмотр истории всех событий
- Автоматическое логирование всех действий


Команды:

```
Команды:
  add <задача>    - Добавить новую задачу
  list            - Показать все задачи
  done <id>       - Отметить задачу как выполненную
  delete <id>     - Удалить задачу
  events          - Показать историю событий
  help            - Показать список команд
  exit            - Выйти из приложения

Примеры:
  add Купить молоко
  list
  done 1
  delete 2
  events
```
