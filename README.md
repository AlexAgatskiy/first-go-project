# To-Do List

Консольное приложение для управления задачами с историей событий.

## Функциональность

- Добавление новых задач
- Просмотр списка всех задач
- Отметка задач как выполненных
- Удаление задач
- Просмотр истории всех событий
- Автоматическое логирование всех действий

##Установка и запуск

```bash
# Клонирование репозитория
git clone <ваш-репозиторий>

# Переход в директорию проекта
cd todo-list

# Запуск приложения
go run main.go

Команды:

add	Добавить новую задачу	add Купить молоко
list	Показать все задачи	list
done	Отметить задачу как выполненную	done 1
delete	Удалить задачу	delete 1
events	Показать историю событий	events
help	Показать список команд	help
exit	Выйти из приложения	exit
